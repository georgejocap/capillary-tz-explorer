Org Timezone
Your company's clock
Set in Organization Settings. Most UI screens display dates in this timezone — what your team sees in InTouch.
Asia/Kolkata · Asia/Singapore · UTC
Server Timezone
Where your data lives
The timezone of the Capillary server cluster. All GET API responses return timestamps in this timezone.
India +05:30 · EU UTC · APAC +08:00 · US −05:00
Input Timezone
Offset you send in
When you POST with an ISO 8601 offset, Capillary stores it. The promotion engine uses it to evaluate time-window rules.
2026-01-23T09:00:00+05:30 → offset stored

Set Org Timezone in InTouch
1
Click the Profile icon (top right) → Organization Settings
2
Go to Organization Profile → click Edit
3
Click the "Set Org TimeZone" tab
4
In Time Zone, add all timezones your org operates in
5
In Base Time Zone, select your HQ timezone — default across all UI modules
6
Click Submit. All UI modules will now show dates in your base timezone.
